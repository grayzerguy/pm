# פיתוח עתידי

רשימת רעיונות ושיפורים מחוץ לסקופ ה-MVP הנוכחי.

---

## אימות משתמשים

- טבלת `users` ב-DB עם username ו-hashed password
- endpoint לרישום (`POST /api/register`)
- login שבודק מול ה-DB במקום credentials קשיחים
- תמיכה בכניסה דרך Google / GitHub (OAuth)
- שחזור סיסמה דרך אימייל

## ריבוי לוחות

- כל משתמש יוכל ליצור מספר לוחות קנבן
- ניהול לוחות: יצירה, מחיקה, שינוי שם

## שיתוף ושיתוף פעולה

- שיתוף לוח עם משתמשים אחרים
- הרשאות: צפייה בלבד / עריכה

## UX ועיצוב

- תמיכה ב-dark mode
- התאמה למובייל (responsive)
- אנימציות ומעברים חלקים יותר

## Backend ותשתית

- JWT במקום session cookie
- rate limiting על ה-API
- logging מובנה
- פריסה לענן (Render, Railway, Fly.io)

## AI

- היסטוריית שיחות נשמרת ב-DB
- תמיכה בעברית מלאה בצ'אט
- בחירת מודל AI על ידי המשתמש
