# Backend Overview

הקבצים ב־`backend/` עדיין צריכים להתבסס, אבל התכנית היא לבנות אותם תוך שימוש ב‑FastAPI ו‑Uvicorn. המנגנון המרכזי יהיה:

* `app/main.py` – יגדיר את האפליקציה של FastAPI, יטען הגדרות מקובץ `.env` ויאפשר קורא API ו־serve סטטי בעתיד.
* מסלולי HTTP:
  * `/` – יחזיר בהתחלה HTML פשוט ("hello world") כהוכחה לחיים, ואחר כך יוגדר לשרת את הבנייה הסטטית של ה־NextJS.
  * `/api/health` – endpoint ייעודי לבדיקה, מחזיר JSON של מצב המערכת.
  * `/api/board` – (בשלבים מאוחרים יותר) לקריאה וכתיבה של לוח ה‑Kanban.
  * `/api/ai` – (בהמשך) יעביר שאילתות ל‑OpenRouter ויטפל בתגובות מובנות.
* חיבור מסד נתונים SQLite שנוצר אוטומטית אם אינו קיים; נדאג לשמור טבלאות `users` ו־`boards` כאשר תוכן הלוח מאוחסן כ‑JSON.
* שימוש בספריית `python-jose` או דומה ל־JWT לפשטות התחברות מזויפת, ולקריאות ל־OpenRouter נשתמש ב‑`httpx` או `requests`.

בצד הבדיקות, יש לכתוב יחידות בדיקה עבור כל מסלול API, כולל הדמיית מסד הנתונים הזמני והרצת בדיקת "2+2" דרך חיבור ה‑AI.

קובץ זה יתעדף עודכן במסמכים נוספים כנגד הקוד הממשי כאשר הסקפתולינג יתחיל.
